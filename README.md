# Fighter Gym

Simple gym website made with Flask.

## Run it
```bash
git clone https://github.com/Ayan1Jawaid1/fighter-gym2.git
cd fighter-gym2
pip install flask
python app.py

Open http://127.0.0.1:5000

Files

app.py → main code
templates/ → HTML pages
static/style.css → styles

Deploy (free)

Go to render.com
New Web Service → pick this repo
Build: pip install flask
Start: python app.py
